📚 Learn Lingo Pet-Project
Learn Lingo is a web application for online language learning, designed to connect students with language teachers and allow booking trial lessons.

The application includes:

- Registration and login for users  
- Personalized favorites list for logged-in users (add/remove teachers)  
- Teachers page with filter by language, student level, and price per hour  
- Teacher cards with detailed info, reviews, and expandable "Read more" section  
- "Book trial lesson" modal with form validation using react-hook-form & yup  
- Persistent state for favorites and interactive elements  
- Responsive design for desktop and laptop screens  
- All interactive elements work according to the technical specification

🧰 Tech Stack
Node.js | Next.js | TypeScript | REST API | CSS Modules | Zustand | Axios | React Hook Form | Yup | React Hot Toast
